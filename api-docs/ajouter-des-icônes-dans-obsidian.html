📖 Ajouter des icônes dans Obsidian

🔹 1. Utiliser les emojis natifs

La méthode la plus simple pour ajouter des icônes dans Obsidian est d’utiliser des emojis.

📌 Comment insérer un emoji ?

  1. Sur Windows : Win + . (point)
  2. Sur macOS : Cmd + Ctrl + Espace
  3. Copier-coller depuis Emojipedia

✨ Exemple d’utilisation

# 🚀 Mon projet Obsidian
- ✅ Tâche à faire
- 📌 Note importante
- 🔥 Priorité haute

💡 Astuce : Les emojis fonctionnent partout dans Obsidian, y compris dans les titres, liens internes, et balises.

🖼️ 2. Ajouter des icônes SVG avec le plugin Obsidian Icon Folder

Si vous souhaitez personnaliser les dossiers avec des icônes, vous pouvez utiliser Obsidian Icon Folder.

📥 Installation du plugin

  1. Aller dans Paramètres > Core Plugins et activer Community Plugins.
  2. Ouvrir Community Plugins et chercher Obsidian Icon Folder.
  3. Installer et activer le plugin.

🔧 Ajouter une icône à un dossier

  1. Clic droit sur un dossier > Change Icon.
  2. Choisir une icône dans la bibliothèque ou ajouter un fichier SVG personnalisé.

💡 Le plugin supporte FontAwesome, Material Icons et d'autres bibliothèques d'icônes.

🎨 3. Insérer des icônes FontAwesome dans le texte

Obsidian ne supporte pas nativement FontAwesome, mais vous pouvez l'intégrer via du HTML et CSS.

✍ Exemple avec du HTML

Ajoutez ce code dans une note :

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css">
<i class="fas fa-coffee"></i> Café du matin
<i class="fas fa-check-circle"></i> Tâche terminée

⚠️ Cette méthode fonctionne seulement avec les plugins qui supportent le HTML (comme Obsidian HTML Export).

🛠️ 4. Utiliser des images comme icônes

Si vous voulez utiliser vos propres icônes, vous pouvez ajouter des images en Markdown.

📎 Insérer une icône image

![Icone maison](images/maison.png)

💡 Astuce : Si vous stockez les icônes dans un dossier spécifique (/icons), utilisez des chemins relatifs.

Méthode Facilité Personnalisation Plugins nécessaires
🟢 Emojis ✅ Très simple ❌ Peu personnalisable 🚫 Aucun
